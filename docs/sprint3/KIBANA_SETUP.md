# Importação de Dashboards e Consultas no Kibana

Este arquivo explica como importar os dashboards e consultas salvas no Kibana usando o arquivo JSON disponibilizado.

## Passos para importar

1. Acesse a interface do Kibana na URL configurada (ex: http://localhost:5601).

2. No menu lateral, escolha **Stack Management**.

3. Clique em **Saved Objects**.

4. Clique no botão **Import**.

5. Selecione o arquivo JSON: `esocial-kibana-dashboards.json`.

6. Confirme a importação. Os dashboards e consultas estarão disponíveis na seção Visualize e Dashboard.

7. Você pode configurar os dashboards para seu uso, salvar filtros e compartilhar.

## Observação

- Mantenha o arquivo JSON atualizado no repositório quando houver mudanças significativas nos dashboards.
